一场游戏一场梦

发表于 2017-06-25 | 更新于 2019-05-04 | 分类于 今天我有话说  |  2,102 字  /  约 7 分钟


一场游戏一场梦…

小时候，我们玩过家家、捉迷藏、老鹰抓小鸡游戏…稍微长大一些，我们玩俄罗斯方块、魂斗罗、仙剑、魔兽游戏…
但是，不知道你有没有发现，父母们总是喜欢骂我们贪玩，不知道学习。
难道游戏是件十恶不赦的坏事吗？
自然不是，他们担心的是，我们迷失在游戏的海洋里，不仅游戏没玩好，甚至还混着过完这一辈子却碌碌无为。

为什么游戏会玩上瘾呢？

你可能听说过很多词，譬如即时反馈、荣誉系统、成就感、社交动力等等，这些都是游戏让人上瘾的机制。但问题是，为什么人们会吃这一套呢？

游戏成瘾，通常被看成是负面现象，但这真不是游戏本身的锅，而是我们对待生活方式的决绝 —— 我们的脑子和身体还停留在了一万年前，但是生活和工作又不得不适应现在的社会。

清华大学历史系刘梦霏自称是一个“潜伏在历史系里的游戏研究者”，她认为，游戏提供的不是上瘾，而是一种可能性。
游戏是让人能够完整地自我实现，让玩家回到集体的潜意识流中，狩猎采集的祖先式的生活方式。那是一种意义完整的生活方式，从而能够填补在当代工业社会中，针对个人而言，断裂的意义链条。

游戏是让我们可以暂时地回到狩猎采集时代的玩意。

第一，在狩猎采集的时代，我们的祖先每天要从住的地方出发，探索周边的环境，采集蔬菜瓜果、打猎，每一次，意外之喜总让他们志得意满。但是，现在不一样了，我们每天上班坐地铁公交，会在指定的时间到达确定的地方；很多工作内容，都是预先计划好的；并且，大量的人，他们的工作也只是为了生计，没有意外之喜。

第二，在狩猎采集的时代，我们的祖先并不追求效率，每天的劳动时间很有限。因为那时候没有冰箱等保鲜措施，多采集食物是没有用的。他们有大量的闲暇时间，和自己的家人或者从小就熟悉的人一块相处。回到现在，每天称自己有大把时间，去玩网络游戏试试，绝壁被骂不上进、没出息！

第三，在狩猎采集的时代，我们的祖先每做一件事，都能直接看到它的结果。现在，几乎你当前做的每一件事，都缺乏直接可见的结果。你看，工作大半年了，才知道有没资格升职加薪。

相反，游戏不正是提供给我们，一万年前熟悉的那种生活方式吗？
在游戏的世界里，有探索的乐趣，有无穷的细节，随时获得成就感；可以进行大量的社交，一解生活中的沉闷；有清晰的即时反馈…怎能不叫人留连忘返？

游戏，不仅是网络上流行的游戏

游戏是未来人类表达自我，创造意义的一个媒介，它是下一代人类文明的基础。
游戏有一种神奇的能力，严肃认真的时候不能玩，但是，玩的时候却可以严肃认真。
游戏不仅是一群人沉迷其中的玩乐，游戏会成为我们新的生存方式。这不是什么黑暗的前途，用得好，它将是我们重返自己精神家园的途径。
只不过，这个进程刚刚展开的时候，我们不免看到眼前的乱象。所以，请不要过于慌张，玩就尽情地玩吧。

游戏，不仅仅是网络上流行的游戏，它早已渗透你我的方方面面。秒杀、搞活动也是商家游戏的一种。
再看学分制的游戏，比如美国的一些游戏化中小学教育，他们把减分制换成了加分制。从小到大，我们总在接受考核，而那些考核往往是总分固定，做错了立马扣分，像极了一个惩罚体系；但是，如果换成加分制，那我们的学习动力显然就没那么变相了，在激励系统的管辖下，我们更早的学会社交互助、合作共赢。

游戏化你的工作、生活

一个工作单位呆久了，很多人要么无聊，要么焦虑。
为什么会产生这些负面情绪呢？因为如果事情太简单了，有大把时间，你会觉得太过无聊；而如果事情太难，又被催促之下完成，恐怕不焦虑都不太可能吧。

游戏化，恰恰可以改善这些不良情绪的。
游戏的一个重要目的，就是制造一种“心流”。

无聊的时候，我们可以对工作的难度提升，譬如一个小时能完成的，今天我们就试试半个小时能完成吗？譬如平时半天写一份报告，那我们试试看，可不可以同时复核完毕发送出去呢？譬如我们平时用键盘随手写程序，这时候是不是可以顺便把思维导图也画一画、文档说明也写一写呢？
这不是没事找事，而是自己到底处在这样的位置，是不是已经足够出众。如果是，公司又不赏脸晋升，你何苦埋汰而不是另择高枝呢？
相反，选择无聊的混日子方式完成工作，难道对于宝贵的一生，你的“良心”真的不会痛吗？

焦虑的时候，我们可以拆解工作的难度。
任务拆解，是游戏中惯有的伎俩。游戏不会一开始就叫你单挑 boss，而是让你去练级，找人组合力量。工作也是一样的，看起来难度略大的工作，你不用以为那是领导故意为难你。其实，恰恰是领导想提升你，并且不会挑一个难到你无从下手的任务交给你，这时候我们可以有焦虑，但是做事情的时候，把任务一项项拆解出来，哪个环节需要谁的帮助，哪个环节需要去哪里找答案，这些都一一罗列出来。
即便最后真没有做成，大家都能看到你的努力和付出，自己也欣慰。

除了工作上的焦虑，我们生活中又何曾少过焦虑呢？
我们常常会在考不好的时候大哭，我们会在表白前坐卧不宁。可是，跳出事件本身，这些都没什么大不了的，只是我们沉浸在失败的后果中，有时候还不免担心被人笑话。
人生如戏。
在游戏里，失败是常态。你看，有人为一场游戏破口大骂的时候，总有人出来劝说，”不过一场游戏而已”。当场气炸的、较真的还不少，他们当真是没分清游戏跟现实的分量？不是的，他们很快就调整过来了。
Facebook 公司有个规定，一旦某个员工任务没完成，就会在他的桌子上放一只可爱的小熊，说明这是惩罚，但是没那么严肃。虽然犯错是丢了点面子，但是大家对于失败并没有那么畏惧。

游戏化的未来

学习的本身是痛并快乐着，但是，如果学习被游戏化，也不免还需要付出些辛劳，那我们会不会更快乐呢？
现代文明的指引下，游戏化进程一步步推演，最终变成人和游戏的未来，那便是更美的将来。

不要再妖魔化游戏了，我们终将有一天被它造福。

你瞧我，一本正经的样子，铁定是不会跟谁玩游戏的。
可是，最后的最后，我还是希望你和我做个游戏，一个一天内不说”我”字的游戏，来一块玩吧？
