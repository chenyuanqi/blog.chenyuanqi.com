写作这件小事

发表于 2017-04-23 | 更新于 2019-05-04 | 分类于 今天我有话说  |  2,519 字  /  约 8 分钟

写作这件小事，大有可为。

在我的世界里，最重要的四项能力指标是：有效沟通，写作，英语 ，编程。没错，这里边一直躺着一个叫“写作”的家伙，它无疑是一把集交流和思考的刀锋之影。
关于写作，除了写日记、文档和笔记外，blog 俨然成为了我额外的一份投资。
每当夜晚青灯初上的时候，深感写作就像是窗外即将淡去的一抹绿色，大写着时光的印记 —— 因写作，故我在。

那么，为什么得空不去嗨？为什么选择苦哈哈的写作呢？
你看，有人内容创业实现了财富自由，有人通过写作圈了不少粉丝。
但是，我要说的是，文字带来的穿透力并不比游戏少。
譬如咪蒙的《致贱人：我凭什么要帮你？！》曾经横穿过内心的一个字 —— “爽”（吾非咪蒙粉）。

其实，写作的意义不在于火急火燎的跟风、出风头，而是一种实现自我表达、升级思维的方式以及打造、传播个人影响力。
至于实现自我表达，说的其实是写作的第一个目标，不为讨好别人，而是表达自己、更懂自我。
又，把语言当作是一维世界，文字是二维的。我们作为一个活在三维里的人，以一二维做基石，站在更高的创作维度，get 到更远处的思想，从而进行思维升级。
然而，毋庸置疑的是，人活着很大程度都是在寻求别人的认可以及自我帝国的强大。那，写作便是你影响力传播最快的方式之一。

进一步来看，根据不同的写作动机，我们对写作也相应的进行了一些分门别类。
寻求自我表达者，可以写写日记、散文以及感悟等，这种文体主要是以写“爽 ”为目标，不需要太在意别人怎么看。而且，自由式写作是一个人开始写作路上的第一环，它就是对自己的长情告白。
升级思维者，记录式的写作，可以写会议记录、工作文档及工作流程等。何以见得这是在升级思维呢？写作是碎片中的结晶体，必能练就一定的思维高度。
追求影响力者，是把一个好观点、好的思想去引导别人观看你所看到的世界，当然那个世界很有可能只是你凭空臆造的。

接下来，我们谈谈如何功利式写作。

就我而言，写作的基础流程三部曲是：构思，写作，修改。
首先，要声明的是，写作之路并非遥不可及。
如果下不去笔，不妨先问问自己这几个问题：

1. 最近有特别想分享的事情或观点吗？如果有，请用一句话表述并记录下来
2. 这个观点有什么不同呢？为什么会让你觉得它有价值呢？
3. 知道了这个观点，有哪些观念需要预先确认的呢？想一想如何使用过去的概念链接它们
4. 如果这个概念成立，接下来又当如何去实践呢？想一想方法论与实践的区别
5. 你的观点是什么？作为一个写作者，怎么能够只是去生搬硬造剽窃别人的成果呢，要有自己的观点，即使它不是很独特，甚至不是很光鲜。

请认真回答如上这些问题，并记录下来。这是写作前的准备，它需要结构化，正所谓“潮水退去，我们才知道谁在裸泳”。
除了下笔难，我们还会遇到另外一位杀手 —— 半途而废。
因为即便是开始动笔了，也不一定能写出个所以然来。写不下去的状况就好比机器故障，你的这次写作分分秒秒被扼杀在写作未长成的摇篮里。
是的，一气呵成的文章只存在梦幻里。
半途而废是可耻的，很大程度上也证明了我们并没能在文字的组合方面循序渐进地做好准备。
那么，请开始你的表演！
从简单处着手，尽可能的去记录你的生活、感悟，在工作中施展文档型写作的招式。
这有点像习得大招前的磨练，也只有这样，通过不断催化自己，最后才能倒逼影响力处大展身手。

我们知道，写作的方式千千万。但是，必须要注意的是，不要让你的文章只有知识点，而没有自己的观点。
要知道在写作这个点上，固执的偏见都比平庸的和稀泥好。

话说回来，写作之路从来就不可能一蹴而就。
写作的最后一步是修改，这是写作最耗时间的一部分，比如我们经常看到很多书籍的第 N 版。甚至在几年后你再看它的时候，都有抽自己嘴巴子的冲动。那，我们的修改大致也有三个阶段：
首先，通篇修改用词的错误，修改词不达意的地方，甚至一些段落太长或太短对整体结构的影响。这是初步的修改，为的是能准确表达你的意思。
进而，做视角的切换。我们通常有两个视角，一个是作者视角，一个是读者视角。在修改阶段，可以设身处地的切换为读者视角，来观察自己看文章时的情绪体验。是的，脱离写作者身份，去看看哪些地方文气不通、表达难度高低不一。这一改，是能把要传达的信号增强的。
最后，是给文字润色，配图或图表。这样，可以实现更高级的表达，毕竟不是谁都有时间来漫游你的文字。这一点，我自己也做的不好，个人比较懒、有些被冷落找不着热情迸发的缘故。只不过，润色倒是可以努力一下。

说了写作的流程，那我们再说说一些写作路上可能遇到的坑坑洼洼。
最值得警惕的坑，莫过于过多的被自己感动，或者过度的打压自己。
感动自己的，常常会有烟雾弹的效果。这个时候，一个好的“品味体系”就显得尤为重要了。但是，你永远不知道别人会不会也跟你一样因此而感动，兴许只是你自己的 G 点太浅，这都说不好。我们可以做的，尽可能设身处地的替读者思考，提高自己的表达技巧。然后，也多想想自己的写作初衷。
然而，说到语义表达，肆意使用修辞或者追求完美辞藻也是一坑。这坑在某些文字大师面前倒还好说，文字功底差些的就要废老大力气了。那么，出坑点是“把写作目的表达清楚”，这才是写作者最该关心的问题。

不得不承认，总有在路上等着你来处理的坑。
只是，最终我们都会想要提升自己的影响力，因而写作莫过于写一些对别人有用的文字。
那就造些问题，以满足我们追逐功利而不辞辛苦的狂妄内心： 

1. 我写的这些对别人有没有用呢？
2. 我写的这些对别人有用的话，有多大的用处呢？如何才能做到最有用？
3. 我写的这些对别人有用的话，对多少人有用呢？如何做到对更多人有用？
4. 我写的这些对别人有用的话，在多长的时段里对别人有用呢？如何才能让那个时段更长？
5. 我写的这些对别人有用的话，如何才能优雅的让别人恰当的反馈给我呢？

如果你能写出大受欢迎的内容，那一定是你“思考能力、分析能力出众”的结果。
相反，投入与产出，可能最终得到的不是自己想要的结果。我们即便不能立即写出爆文，但也总可以通过写作习得一些关于学习、思考、分析的能力，它们总能在你的文字世界里徜徉出自己的模样。

写吧，只要真下手去写，就只能真心思考了。
你会变成一个勤劳的信息采集者，整合信息的能力会让你越来越像一个组织者，实践以产出，思考以前行。
不得不提的是，写作确有天赋因素的存在。有天赋也固然是件好事，但是如果仅凭天赋的有无而论断这辈子该不该做哪件事情，那将失去多少有趣的事情呢？ —— 哪怕，在我看来，写作在很多时候也只是一个人的狂欢。

写作是件小事，总能带给你惊喜；
悲伤就像是谷仓里的米，每天都会减少一点；
把幸运的人扔到河里，他会叼着钱爬上来；
都说人是细胞和血液，其实人更像叶子，
只要风经过，他就会唱歌 …

最后，泼点冷水吧，这样会更清醒：虽然写作这件小事大有可为，但并不意味着所有人都需要它。
唯有把写作放在一个正确的位置，不因潮流，不因变现，不过分期待，那样才不至于逐渐迷失写作最初的素颜美。你需要的仅仅是写作，所以写作。
如果你最终跟我一样选择了投资写作这件小事，真幸运，你我都在努力把控自己的生活。
