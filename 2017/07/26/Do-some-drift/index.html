一点心路，两点漂移

发表于 2017-07-26 | 更新于 2019-05-04 | 分类于 今天我有话说  |  776 字  /  约 3 分钟


今天，简单谈谈个人最近的一些想法。

记不起“小肚子”何时来的，只知道这已然好几个月的“身孕”，一直没见有大动静。
可是，最近空了大半年的隔壁房间，忽然就多了几个人。这动静未免有点大了。当一个人住一个楼层住成了习惯，一下子来了 4 个人，就像来了外敌一般，略有不安。一来不能随意大声歌唱了（虽然唱得不好），二来出入门都可能碰到的人当如何相处。

我们先说说运动这事儿。
每周的周末，骑共享单车的活动已经有段时间了，大概是廖兄过来一起去骑了一回之后吧。说到底，也就是没什么活动又不下雨的时候，果断骑车是个不错的选择，哪怕只是一个人。
相比去附近校园的操场跑跑步，骑车是更有趣的。它不会让你感觉太累，甚至可以抽空去看看沿途的风景，呼吸着新鲜的空气，看看小孩子们玩沙子…能做很多很多的事情。然而，让我头疼的是，多走几个分岔路口就真不知道是什么方向了，手中的破手机恰巧没电的时候，那就真 gg 了！所以，也就不能骑得太远。
当然，光周末这点运动量怎么可能算减肥呢？我有每天早上做些俯卧撑、仰卧起坐之类的，只希望身体不要太早垮掉。你瞧，大沙河公园里，有多少人起早跑步锻炼，我真不是一个人在战斗！

再来说说关于邻居的事情。
在外头的邻居真不比家里边的邻居，即便都把租房当成了自家来摆设。我是不愿发出声响去影响别人的，但是却很反感别人的大声喧闹，尤其睡觉期间。于是，我害怕跟邻居熟络之后，更随意的被打搅。
我总觉得，人和人之间的关系除了亲密无间、形同陌路，还应当有相安无事。
总是听说“远亲不如近邻”，可似乎总有个条件在左右着 —— 那也得是优质的邻居吧。所以，也只希望各自安好便罢了。

说到底，每个人又何尝不是有各自的选择。
就像周末除了骑骑车，更多的是感谢这些生活里所遇的便捷。于是，更喜欢在外面漂移，也就更像一只漂流瓶，把它扔向大海，永远都不知道下一个打开它的人会是谁。
也像对待邻居的态度，更多的是找到了为生活奋斗的意义。于是，即便有时候游戏里飞车漂移，也就更像速度与激情，把它拍成影片，永远也不知道何时才是大结局。

纵笔一旁，美美生活，取些夜色，填满心田；
谷雨凉夏，说说心路，来点漂移，如此甚好。
