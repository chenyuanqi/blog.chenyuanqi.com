从兑换货币这件事情说起

发表于 2017-11-19 | 更新于 2019-05-04 | 分类于 今天我有话说  |  947 字  /  约 3 分钟


前几日，去中行兑换印尼盾了，因为近期有这方面的需要。

然而在我眼力，兑换钱这种事情，一直以为只是一手交钱另一手也交钱这样就能完事的。但是，事实上显然不像黑市的交易，银行必定要走它的流程。那就走流程吧，提前周末去做预约就好了。然后，知道下周二才能取。那也无妨吧，毕竟还没着急到必须预约当天就拿到。
直到要取的那天，天且还尚早，银行门口已经陆陆续续地排起队伍来了。我暗想，原来有相当一部分的人不是一定在自己的工做岗位上，我是不是坐久变傻了？

我时不时地看看时间，终究这里距离上班的地方还有些远。始终觉得，要保证既取得了钱又不用请假，可是银行的办事效率并不如期望中的那般，说要等半个钟没准就一个钟。
眼下算计一番，真等上半个钟，那也必然是要迟到了，我只好跑去公交站。可是犹豫了，到底该不该请假，到底还来不来得及赶过去公司。
索性，快速计算一番。办完有时间就叫的士应该赶得上，没时间就请半天假。

最后取完钱之后，还有大约半个钟。打的应该是完成没问题了的。
滴滴，一路上，司机都在卖弄自己的见识，却不顾我焦急的眼神。他告诉我地图其实可以用手拉大，然后定位就准了 … 一直到了半路，才发现有交警叔叔 … 这回可真真是糗大发了，司机慌忙叫带安全带，却不料被逮个正着！车子被拦了下来，我傻傻的望着，心默念：这真是越想快就越有阻力啊。自然按照规定，司机被罚了 200 块钱，而我也恰恰是要迟到了 。。。
回到车里，一脸不满的司机说，你看这怎么办吧？
我也没多想，一人一半呗。是啊，说到底，谁打工容易？更何况这是由我导致的罚款。我递过钱，下车，然后转身就奔向公司。

我已然忘记司机的窘迫。
后来，我甚至觉得，那其实完全可以不赔钱，只是良心真的不会痛吗？
相比于钱财，更让我不安的是，我信条里罗列的“若打车，务必戴好安全带”都去哪里了呢？我是不是在不知不觉中把自己给卖了？

啰里啰嗦的写下这些，简单来看，其实就是一件小事里的小波澜。
细想，这生活又哪里有那么多大事情值得烦扰，都不过是一件件小事没处理好的结果罢了。
那，我为何一定要兑到钱，兑到钱也不会得到谁眼里的一丝丝感激。那一刻，心冷的想起有人第一次跟我说”喂狗”而自己浑然不知是何滋味！直到现如今，其实喂狗都不重要了的，毕竟谁做事总能希望别人怎样，只不过求一分心安而已。

总之，处理事情总有高手，在事情的背后绕个弯，给自己一点时间，状态也终究可以调整回来。
我期望的是，即便每天必然要做的锻炼都可以不做，甚至玩忽职守的荒废掉一天，但是在空闲下来的时候，不应该只是懊悔乃至不安，而是想想还有没有更好的活法，又如何才能活得更好，那才是蜕变的起点。
