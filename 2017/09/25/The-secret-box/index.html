高手的神秘暗箱

发表于 2017-09-25 | 更新于 2019-05-04 | 分类于 今天我有话说  |  868 字  /  约 3 分钟


面对呼啸而至的时代车轮，我们必须加速奔跑。有时会力不从心，有时会浮躁焦虑，但必须适应。它可以轻易地将每一个落伍的个体远远抛下，辗作尘土，且不偿命。 —— 《在轮下》

时代的车轮滚滚，联机学习是高效学习者的神秘暗箱。
那么，什么是“联机学习”呢？联机学习，就是指什么事都不懂，然后去找那个领域最牛的人聊天。

同样的，在这个时代，知道知识在哪，比知道知识是什么更重要。
知识铺天盖地，搜商应运而生。有时候，真心觉得脑容量不够，遗忘的速度甚至吓到自己。还好，信息的记忆不是最重要的，做创造性的事情才更重要。
那么，获取知识的途径，可以从一些大咖如古典，俞敏洪，李笑来，罗胖，成甲等人书籍阅读，app 或者订阅号也是不错的途径。
当然，除了阅读的方式，网上搜索、群友交流、听课都是不错的途径。

纵观如今的每日匆忙，我们的生活往往就在两点一线之间辗转。回头看看的时候，真害怕自己什么都没学会，就度过了大半辈子。
是的，慢下来，可能更快吧！而我们所说的慢下来，并不是什么都不做，而是想清楚了再去做，这样反而更有效。就像某句话所说，方向比努力更重要。因为方向错了，再努力，也不过是多做了些无用功罢了。

回顾历史。
在 1825 年火车刚刚试车成功的时候，这个又笨又大的铁家伙，其实它所遭受到的冷眼比赞扬要多得多。每次火车开出去的时候，总有好事的农场主驾着马车与火车赛跑，而且总能赢过火车。比试完之后，自然就可以跟众亲朋好友们举杯相庆，互相调侃火车一番。
殊不知时间辗转而来，再也看不到马车追逐火车，因为再也追不上了。
其实，火车的发展，恰恰说明了一点：可怕的不是优秀的人比你努力，而是优秀的人有更好的办法让自己迭代。那些只会在旁指手画脚的人，最终只能羞愧难当。

有时候，常常在想，人要有多努力，才能看起来毫不费力？
在努力的同时，我们也对努力思考，甚至想想高手如何炼成。进而，如果天才画家拉斐尔使用过投影仪，那我们是否会认为各个领域的高手都有自己的“暗箱”呢？

相对学习来说，学习本身就应当是能解决当下的一些问题，带着问题去找答案的效果最好。再通过和不同行业的人进行交流拓展视野，尽量颠覆自己的认知，尤其要关注那些和自己观念完全不同的观点和理论。进行思考和逻辑推导。

高手的神秘暗箱，自然不是耗费百倍的努力。
每个时代的高手们都在利用社会和科技的底层逻辑撬动自己，实现跨越式的发展。
万物互联的红利，带上神秘暗箱，继续前行。
