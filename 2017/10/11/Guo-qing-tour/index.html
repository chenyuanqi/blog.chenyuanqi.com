进化菌的国庆之旅

发表于 2017-10-11 | 更新于 2019-05-04 | 分类于 今天我有话说  |  786 字  /  约 3 分钟   |  


既然有早读菌，前端菌，二次元菌 … 我想，我应该就是进化菌了吧，最懒的那个。

是的，简直要懒成狗了呢。
放假这些天，慵懒的一逼。
十天的假期，花一天一夜的时间回家，一天半夜回深，其余时间几乎都在家里“闲游”—— 就是那种什么都不做的，逛荡罢了。
国庆，可谓是举国欢庆的日子呢，如果不是法定假期少，谁会挤在这个点回家呢？

说来也怪，在家的第一夜，我居然果断约见了蓉儿，当时还跟家里人说是去找朋友喝茶的。实际上，毕业的这些年，我们都没有再见过，于是说晚上见就晚上见。
只是，没见到的时候，总是会想起她初中毕业时的留言：希望能多看到你阳光的一面。是啊，好像我才是你要找的“静哥哥”呢，哈哈！
可是，后来的后来，大学再见到的时候，妳却变得脏话连篇。当然，谁也不知道那些年都经历了些什么 …
能再见，就好。

再过几天，也约见了同样老的同学阿民。
风雨交加之间，或曾被困“九层妖塔”下。
有时候常常在想，到底这个世界也太特么的奇怪了，冥冥之中就有些人凑在一块，而怎么也想不明白那是为甚。

渐行渐远的日子里，珍惜也许都会幻化成为妄念，可谁会在乎这些呢？
约不约，日子也会一天天过去。

反而，在家的时候，我的进化信仰索性就蜕变成了最懒进化，可能那才是我的本性吧 … 就那样，倍感身心疲惫，累得只剩下睡觉的想法；不想出门，因为出门也不知道有什么值得去做的事情。

总之，还是老样子吧，不想知道家里面的一切。天然的以为改变不了任何现状，所以只是维持住自己的原本模样，所以更像是漠不关心的异类。
至于家庭的矛盾，我偏执地以为万物皆有解。简单来说，都是闲的慌致使；往深处看，其实处理复杂关系中，试问谁又能轻易得到解脱？我不想置身其中，甚至不愿早些踏足纷扰 —— 自己都活不好，何必还要跟着起哄、制造困境呢？

回到自己的狗窝，放下行囊，感觉一切如从前般轻松。
从冰箱里拿出一瓶奶，喝上几口，然后疾书而弛，那才是我现在的生活。

平平凡凡，进化菌的国庆之旅。
回深的时候，咳嗽还咳得异常厉害，有时候半夜难受也睡不好 …

那，就简单写写好了，因为 2017 的末端还有很多事情要着手。
自然，有人在这段日子里躺着安然度过，也有人已经偷偷的抢跑了 … 那么，你呢？是否也终将会和我一样，放下慵懒的姿态，继续前行。
