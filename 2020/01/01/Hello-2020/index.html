你好，2020

发表于 2020-01-01 | 更新于 2020-01-01 | 分类于 2020  |  1,207 字  /  约 4 分钟


时间已经来到了 2020，这是一个新的开始。

尝试着回忆一番 2019，奈何记忆力太差，甚至连当初的计划都找不着了。

还是简单回忆一下 2019 吧

不管怎样，简单回忆一下是有必要的，即便做不了多好的总结。

减肥的情况还是没有彻底落实。
现在想想，减肥才不是我最重要的事情，去除“小肚子”才是！有时候，看看自己所谓的“轻断食”，反弹起来着实可怕，稍微多吃几天的晚饭就能让体重恢复如初！减肥其实还是需要加上一些运动，甚至生物钟的调息。

女票还是个计划。
这个年纪里，成家几乎都天天挂在眼前了。但是，这也几乎不是能着急的事情，也许还得看看天意，挑选个黄道吉日，然后遇到她。
今年也算做了些改变吧，接受家人的介绍，尝试去多和别人网上聊，尝试面对面聊，尝试 … 做了很多尝试，只是尚未打破僵局而已。

Golang 也还是没用上。
虽然年中的时候，有机会研究一波「微服务」，最后还比对了另一个语言 —— Java。不幸之下，花了将近一个月的时间在试探着深入 Java。最后，Golang 就被冷落一旁了呢。

除了一大堆没实现的东西，难道就没有一丝丝做成的小事吗？
自然有。譬如读了两遍吴军老师的《态度》，写的真真是棒极！譬如做汤做菜越来越好了；譬如去云南，然后心生出行的喜悦 … 这一年里，还偷偷换了个稍微安静些的住处，重置了家当；虽然小家碧玉，好歹心安处即吾家。

2020，躬身入局

曾国藩曾经讲过这样一个故事。

农村里，有个人出门，看到在一条很窄的田埂上，俩人顶上了，谁也不让谁，谁也过不去。为什么不让呢？因为俩人都挑着很沉的担子，路太窄了，谁要让，谁就得从田埂上下去，站到水田里，沾一脚泥。
有个旁观者就走上前去说，来来来，我下到田里，你把担子交给我，我替你挑会儿，你一侧身，不就过去了吗？

是的，你作为一个旁观者，想上去劝，咋劝呢？谁也不服气啊。但是，只要你的身份稍稍转换一下，从一个旁观者，变成一个置身其中的人，把自己放进去，是不是一个看似无解的事，就有了答案呢？
这就是“躬身入局”，把自己放进去，把自己变成解决问题的关键变量。

上面已然罗列了一些 2019 错过的事情。
与其说是错过，不如说是没有很好的规划，换个词就是「没有使用 OKR」。那么，下面就用 OKR 开启 2020。

减肥改成去除“小肚子”，今年 3 月份完成。
也就是需要用一个季度的时间去完成，那具体的 KR 有哪些呢？

[] 早睡早起，23：30 必须躺下，5：30 必须起床
[] 起床洗漱后，执行运动：卷腹 20 + 俯卧撑 30 + 深蹲 30 + 跳绳 100
[] 每日步行数量：不低于 5000 步
[] 食有规律，尽量多吃蔬菜瓜果少吃蛋白、肉类，20：00 以后不能再进食

婚姻还在路上。
相比家里人催促，自己反倒更认可先把自己过好。而如何定义这个所谓的“先把自己过好”呢？都有哪些具体的 KR 呢？

[] 深度认识自我、本我与超我
[] 热爱生活，取悦自己，哪怕只是折一只千纸鹤寄去思念
[] 每个周末做一道自己没做过的菜，享受庖厨之乐
[] 好好学习摄影，极客时间 App 里有相关的课程，重在实践
[] 国庆抽空去一趟西安

多学点理财的事情。
其实，理财才是一切的基础。有钱真的可以解决大部分问题，显然理财是跟钱最相关的事情。当自己没有工作的时候，时常在想，怎样才可以赚到除了主业工作之外的收入呢？

[] 做好主业，做个靠谱的员工
[] 继续每月定投「蛋卷基金」
[] 通读百度云盘的理财资料，提升理财思维
[] 关注某宝等的褥羊毛活动，想办法放大收益
[] 关注财经，尝试付出劳动换取收益

就这些吧。
那些乱七八糟的诸如戒烟戒酒、不要久坐 … 的恶习，也要赶日程上邢台了，总不能还留着过春节吧？

你好，2020
