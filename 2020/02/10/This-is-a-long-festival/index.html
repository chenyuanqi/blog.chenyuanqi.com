这个春节有点长

发表于 2020-02-10 | 更新于 2020-02-10 | 分类于 2020  |  1,102 字  /  约 4 分钟


一直觉得 2019 让很多人觉得很南、超级南，没想到 2020 只用了段段一个月就把 2019 给比下去了 —— 全民南。

一场名叫“新型冠状病毒”的疫病，让整个天朝的大江南北都陷入了空前“封锁”中。
有人恐慌，有人奋起抗击，有人赚足了灾难钱 … 云云众生，我想更多的人会选择“不出门，就是为国做贡献”。

原本没有打算回家

回想起春节前，自己甚至还对回不回家这件事情心存左右为难的时刻。只因为想起父母的花白双鬓，这才下定了决心。
那时候，确实已经接近春节，疫病已经开始全网散播了 —— 我猜在深圳应该是人尽皆知的程度了，而菜市场、餐馆甚至车站都有大把的人没戴口罩，有时候不由得叹服：人心是真的大啊！

回家“封锁”了好几天，村里开始需要每户派人去看护村口，防止村外的人进入。那时候，我们劝父亲戴口罩，父亲尚且有些不屑，最终还是戴了。有人说：戴口罩就能不被感染吗？
自然没有绝对的可靠。只是从概率学上看，戴口罩被感染的概率更低。我们总觉得命运是老天爷给的，但是如果漠视概率而选择任由天意，想必老天爷也会欢迎各位光临那个所谓的“天堂”。

原来我们的人类也很脆弱。
看看“鼠疫”、“霍乱”等那些惨绝的疫病历史文章，你会看到，其实我们能做的不多。一场疫病足以让人类重伤，甚至濒临灭绝！

好好呆着，也许一切很快就好

呆在家，就调整好心态吧。
既然现在不能出去，就要接受这个现实，而不是一味地抱怨，抱怨越多，运气越糟糕。把心态调整好，一切都顺了。要想想自己现在还能在家里待着不用被隔离，已经是非常幸福了，所以这个时候心态一定要放好，家里呆着就行而不是觉得家里呆着闲得无聊要到处跑。

这次特殊情况不能出门，那我们就换个思维。
所有的限制都源自思维的守制，所有的改变都源自思维的转变，所有的奇迹都源自思维的升级。

发生这样的事情对我有什么好处？

呆着不代表无聊。
这样，我们是不是可以静下心去学习，是不是可以好好陪陪家人，甚至可以打打游戏，追追剧，补个觉 … 一个月这么过，这绝对是人生最难忘，最奢侈的春节！

从中得到了什么样的念想？

很多小公司，甚至因扛不起负债而宣告裁员、破产。
但是，我们看到更多的是线上远程的服务，他们流量暴涨。以前已经心心念念的远程工作，没想到这么快就成了大家的刚需。

显而易见的，仍然有很多人只能在家待业，既然不能出去上班挣钱，那就想办法有没有可能在家里挣钱，为什么不能在家里挣钱呢？

要怎样做才能得到好的结果？

如果这次呆在家是 1 个月，3 个月，6 个月，甚至更久，还不能出去上班呢，该怎么办，怎么找到出路？

在家学习，在家办公，在家锻炼 … 我们可以做的事情很多的，有时候甚至还觉得时间不够用呢。
我是这样做的。开始用自己的编程语言，为自己写一些自动化脚本，不仅对工作没落下，还提升了未来的效率；开始学一门新的技术，用新技术写个小工具，服务更多的人 …
相比整天无所事事，恐慌，传谣 … 好好为自己做升级，没有比这更值得做的了！

这个春节长的有意义

这个春节是真的有点长了。
恰好可以多陪陪我们的亲人，恰好我们可以做些自我升级的事情，恰好我们可以尝试新的工作方式 … 这一切，来的刚刚好。
